# 2.基本语法
```
selector { 
	property: value ;
}

选择器 {
	属性: 属性值;
}

```
2.1 CSS的四种引用方式<br />**1.行间样式: 应用内嵌样式到各个网页元素.(嵌入式样式)**
```html
<p style="color:red;margin-left:20px">这是P标签</p>
```
**2.内部样式: 直接在文件内部,head里面定义样式**
```html
<head>
	<style>
    p{
    	font-size:100px;
    }
  </style>
</head>
```
**3.外部样式:     (1)先创建一个CSS文件;     (2)再用link标签引入这个文件**<br />**4.导入外部样式:    (1)先创建一个CSS文件;     (2)在style标签中用import导入这个样式文件.**<br />
<br />以上4种CSS引用的区别:<br />行间样式只作用于当前标签;而内部样式作用于当前文件;外部样式可以被多个HTML文件引用.<br />在实际项目开发中,最多使用外部样式[**link引入**和@import引入两种方式]<br />[1.@import只能加载CSS,而link可以加载其他事务;<br />2.link引用CSS时,在页面载入时同时加载;而@import需要页面网页完全载入之后再加载;<br />3.link是XHTML标签,无兼容问题;@import是在CSS2.1提出的,低版本浏览器不支持;<br />4.link支持使用JavaScript控制DOM去改变样式;而@import不支持]<br />

# 4. 选择器分组与样式继承
选择器分组: 让多个选择器(元素)具有相同的样式,一般用于设置公共样式;
```html
        /*选择器分组*/
        h1, .box, p{
            color:red;
        }
```
样式继承: 子元素可以继承父元素的样式;  
```html
 /*样式继承*/
        .test{
            font-size:18px;
        }
        .test span{
            font-weight: bold;
            font-size:12px; /*改写父元素传过来的样式*/
        }
```

# 6.CSS字体

1. font-size:字号(px/%)
1. font-family:字体()
1. font-style: 文字样式(normal/italic/oblique)
1. font-weight: 加粗(normal/bold/bolder/lighter/100-900)
1. line-height: 行高(px/数字/em等)
1. color: 文字颜色(red/rgb(r,g,b)->r(红):0-255,g(绿):0-255,b(蓝):0-255; /16进制[以#开头,后跟6位(#rrggbb)或3位(#rgb)16进制数])[**推荐直接使用16进制,减少渲染消耗]**
1. text-decoration: 文字修饰(normal/underline/overline/line-through)
1. text-align: 文本对齐方式(left/right/center)
1. text-transform: 字母大小写(upercase/capitalize/lowercase/none)
1. text-indent: 文本缩进(px/em/%/pt)

font复合属性:<br />font: font-style  font-variant  font-weight  font-size/line-height  font-family;<br />**注意:**<br />**1)属性值的位置顺序**<br />**2)除了font-size和font-family之外,其他任何一个属性值都可以省略**<br />**3)font-variant: normal/small-caps[让大写字母变得小一些]**<br />**

# 7.CSS背景

# 9.CSS属性和关系选择器
**属性选择器:**<br />[属性名]:包含有指定属性名的元素<br />[属性名=值]:属性名的值为指定值的元素<br />[属性名~=值]:属性名的值包含指定值的元素<br />[属性名^=值]:属性名的值以指定值的开头的元素<br />[属性名$=值]:属性名的值以指定值的结尾的元素

**关系选择器:**<br />1)空格: 后代选择器<br />2)>: 只选择儿子元素<br />3)+: 兄弟选择器

# 10.CSS伪元素
1)伪元素和伪类的区别:<br />伪类:用于已有元素处于某状态时,为其添加样式.它只有处于dom树无法描述的状态下才能为元素添加样式.<br />伪元素: 用于创建一些不在文档树中的元素,并为其添加样式.比如可以通过:before 来添加在一个元素前增加一些文本,并为这些文本添加一些样式.虽然用户可以看到这些文本,但这些文本实际上不在文档树中.<br />**伪类和伪元素,都不在文档树中;**<br />**伪元素和伪类都是大小写不敏感的;**<br />2):before/:after/:first-letter/:first-line: 前面可以是1个冒号也可以是双冒号;<br />::selection/::placeholder/::backdrop: 前面只能是双冒号.<br />

# 11.CSS浮动布局&盒模型
1.浮动: float,让块级标签不独占一行.目的:把块级标签元素可以排在一行上.[属性值: none/right/left/inherit]<br />2.浮动的原理:  就是让元素脱离文档流,不占用标准流.<br />3.浮动后,后面的元素,不管是块级还是行级,不会显示在下一行.<br />4.清除浮动: <br />目的:让后面的元素自动掉到下一行.<br />方法: <br />1)添加空标签,并设置样式: **clear:both**;<br />clear:left;  清除左浮动<br />clear:right;  清除右浮动<br />clear:both; 清除左右浮动<br />clear:none;   不清除浮动<br />2)在要清除浮动的父级添加样式; overflow: hidden;[超出部分隐藏,也可以用来清除浮动]<br />3) 在要清除浮动的父级添加伪元素,并设定样式:<br />父元素:after{<br />content:'';<br />display: block;<br />clear: both;<br />}<br />**注意在实际项目开发中,我们一般首选第2种方案**<br />**<br />5.CSS盒子模型<br />每个元素都是一个盒子,一个盒子由**margin**(外边距),**border**(边框线),**padding**(内边距)和**contnet**(内容)组成.<br />IE的盒子中的**content**又由**padding**和**content**组成.<br />区别外边距和内边距是以边框为参照.<br />系统默认外边距为8px;<br />1)外边距(**margin**): 指元素边框线之外的距离.<br />margin-left: 左边距<br />margin-right:右边距<br />margin-top:上边距<br />margin-botom: 下边距<br />margin: 可以来设置任意一个边的边距,可以带1~4个参数;<br />1个(apx): 表示上下左右都是这样的外边距;<br />2个(apx bpx):表示上下左右为apx,左右为bpx;<br />3个(apx bpx cpx): 表示上为apx,下为cpx,左右为bpx<br />4个(apx bpx cpx dpx):表示 上apx,右bpx,下cpx,左dpx;<br />2)内边距(**padding**): 指元素的文本内容与边框之间的距离<br />它的用法与**margin**完全一样.<br />3)边框(**border**)<br />**border-width:** 边框线宽度<br />**border-style**: 边框线样式<br />**border-color**: 边框线颜色<br />复合写法: <br />**border**: **border-width** **border-style** **border-color**;<br />注意:**border-width** **border-style** **border-color**这三个参数没有位置之分.<br />
<br />4)盒子的真实尺寸<br />盒子宽度 = **width **+ **padding**左右 + **border**左右<br />盒子高度 = **heigth **+ **padding**上下 + **border**上下<br />
<br />5)**display**属性: 用来设置元素的显示方式<br />属性值:<br />**none:** 不显示元素<br />**block:** 块显示,在元素前后设置换行符,目的是将行级标签转换为块级标签(因为行级标签不识别宽高,而块级标签识别,转换后,行级标签也可以设置宽高了)<br />**inline**: 行内显示,将块级标签转换行级标签,[宽高自适应,若无文本,则不显示宽高]<br />**inline-block**: 将块级或行级标签转换为行内块级标签.[转换后两个元素之间会有一定的间隔]<br />
<br />6)**table**样式<br />table一般不用来布局,主要用来格式化数据.<br />属性:<br />width: 宽度<br />height: 高度<br />border-collapse: collapse; 单线边框<br />border: 边框线<br />td,tr属性:<br />width: 宽度<br />hright: 高度<br />border:边框<br />text-align:文本左右对齐(left(默认)/right/center)<br />vertical-align:文本垂直对齐方式(top/middle(默认)/bottom)<br />7)CSS列表样式<br />不是描述性的文本的任何内容都可以认为是列表.比如:菜单,商品列表等.<br />1)列表类型:<br />1)无序列表(ul)<br />2)有序列表(ol)<br />3)自定义列表(dl)<br />ul和ol的列表都是li表示的,而dl都是由一个dt和一个或多个dd组成.<br />dl一般用来设定一个定义,比如名词解释等.dt:标题,dd:描述,用来对dt的内容进行解释说明.<br />2)样式(用来修改标识类型)<br />list-style-image: 用图像表示标识<br />list-style-position: 标识的位置(inside/outside[默认值])<br />list-style-type: 标识类型<br />a)无序<br />disc/circle/square<br />b)有序<br />decimal(默认)/decimal-leading-zero/lower-roman/upper-roman/lower-alpha/upper-alpha/<br />lower-greek/lower-latin/upper-latin...<br />有序和无序,都有none.<br />
<br />简写:<br />list-style: list-style-image list-style-position list-style-type;<br />list-style的值可以按任意顺序列出,而且可以任意省略,只要其中一个值,其他的都会默认.<br />
<br />8) 轮播图<br />作用:主要用于产品展示或公司相关宣传.<br />组成: <br />1) 轮播的组图(至少两张,不能太多)<br />2) 控制器<br />3) 计数器<br />
